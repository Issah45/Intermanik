## Controls
Arrow keys to move, z to jump, and x to dash.\
dashing can be used to deflect an object\

## Level Editor:
m: confirm tile\
n: place spike\
b: place Manikom\
v: reposition Manikom\
r: remove last tile\
k: remove last spike\
q: place EOL\
s: save\
a: load\
d: add dialog\
l: add lava ball\
